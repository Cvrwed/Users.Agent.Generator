# Info

It is a simple user agent generator using the lib of [UA](https://github.com/iamdual/ua-generator)

The program will infinitely print several **User Agents** and save them in **useragents.txt**

## Use

Windows:
```
$ python uag.py
$ py uag.py 
```

Linux:
```
$ python3 uag.py
```
